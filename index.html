Fruit Classification

The fruit industry is a billion dollar industry producing tons of fruits yearly however in grocery stores, unsold fruit deteriorates, rots, and is subsequently thrown out. I along with five teamates proposed a machine learning model that can identify if fruits are fresh or rotten so that in practice, the model can be implemented in stores, detect fruit as its going bad, and this fruit can be donated to food shelters. The proposed solution is a pretrained ResNet50 model that uses transfer learning and was trained on six different fruits. The overall accuracy result for the model was 98%.

CCTV Footage Anomaly Detection

Anomaly detection using CCTV footage is an important problem to address using machine learning solutions as doing so can identify and report suspicious activity therefore increasing public safety. Many home security cameras do not currently use AI to notify owners, and instead only detect movement. Using the UCF Crime dataset and a Video Vision Transformer (ViViT), a model was made that could classify regular and irregular activity with up to 67% accuracy. Further work is still being done to improve and stabilize the results.
